    • A static site generator that builds the front-end documentation site.

      • Algolia Search: Enhances search functionality by indexing your documentation and returning fast, relevant results. It works by:
        • Crawling your documentation site to create an index.
        • Using this index to provide accurate search results in real-time.
        • The integration involves setting up an account with Algolia, configuring their crawler to suit your site, and updating your site with the necessary API keys and configuration files.
